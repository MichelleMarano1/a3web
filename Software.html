Software and Applications

What computer programs will you need to run?

PCI Geomatica

PCI Geomatics is an industry leader in remote sensing applications. PCI Geomatica offers a suite of programs pertaining to remote sensing, raster calculations, geometric correction, and digital photogrammetry.

In Remote Sensing, you will utilize PCI Geomatica to:

  • Examine spectral properties of land cover and classify landuse
  • Geometrically correct historic sattelite imagery using automated and semi automated approaches
  • Apply various image filters to maximize data quality

ESRI ArcGIS Pro and ArcMAP

ESRI boasts some of the most comprehensive GIS platforms in the biz. A pioneer in GIS, the ArcMap suite has forged the way for modern GIS. Now, with the introduction of ArcGIS pro, ESRI is poised at the forefront of GIS.

In Spatial Analysis, Database Principles, and Geovisualization, you will utilize ArcGIS to:

  • Construct visually striking maps with advanced techniques
  • Conduct meaningful analysis of spatial trends using vector and raster data
  • Create geospatial data from Survey123 for ArcGIS Online

Adobe Illustrator and Avenza MAPublisher

Adobe illustrator is a fantastic option for graphic design and Avenza MAPublisher integrates GIS tools seamlessly into the Adobe Illustrator environment. For creating cartographic outputs, MAPublisher is tops.

In Geovisualization, you will utilize Illustrator and MAPublisher to:

  • Visualize spatial data for general purpose and thematic maps alike
  • Implement aspects of design, including colour, scale, and text
  • Create large format maps with visually appealing design using conventional cartographic techniques

Python

An open source programming language, Python has seen massive growth in recent years as its simple syntax increases readability and efficiency. Python programmers are in high demand as the versatility of python does not go unnoticed.

In Problem Solving and Programming, you will utilize Python to:

  • Automate complex GIS tasks using the arcpy library or standalone scripts
  • Learn the fundamentals of object oriented programming
  • Design practical programs for use in GIS applications

Microsoft Access

Microsoft's legacy database management tool provides a great opportunity to learn the fundamentals of database design principles.

In GIS Database Principles, you will utilize Microsoft Access to:

  • Review data by performing queries using SQL
  • Integrate databases with different structures and design your own database
  • Manage databases concurrently with peers to organize information efficiently
